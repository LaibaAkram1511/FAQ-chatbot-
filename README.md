# FAQ Chatbot using NLP

## Project Overview

This project is an FAQ Chatbot developed using Python, Streamlit, NLTK, and Scikit-Learn.

The chatbot answers user questions by finding the most similar FAQ from a predefined dataset using Natural Language Processing (NLP) techniques and Cosine Similarity.

## Features

* FAQ Dataset Management
* Text Preprocessing
* Stopword Removal
* TF-IDF Vectorization
* Cosine Similarity Matching
* Confidence Score Display
* Chat History
* Streamlit User Interface

## Technologies Used

* Python
* Pandas
* NLTK
* Scikit-Learn
* Streamlit

## Working Process

1. Load FAQ dataset from CSV file.
2. Preprocess the text.
3. Convert questions into TF-IDF vectors.
4. Accept user query.
5. Calculate cosine similarity.
6. Retrieve the best matching answer.
7. Display answer with confidence score.
